# Contributing to CodeMaster

Thank you for your interest in contributing to CodeMaster! We welcome contributions from everyone.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/codemaster.git`
3. Install dependencies:
   ```bash
   npm install
   ```
4. Create a new branch: `git checkout -b feature/amazing-feature`

## Development

Run the development server:

```bash
npm run tauri dev
```

This will start the Vite server and the Tauri window.

## Pull Request Process

1. Ensure your code follows the existing style
2. Update documentation if necessary
3. Open a Pull Request with a clear description of your changes

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
